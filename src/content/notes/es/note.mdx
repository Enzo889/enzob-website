# { frontmatter.title }

JavaScript ES6 introdujo varias funcionalidades nuevas que han hecho que programar en JavaScript sea más eficiente y agradable. Exploremos algunas de ellas.

## Arrow Functions 

Las funciones flecha proporcionan una nueva sintaxis para escribir expresiones de función. Son más concisas y enlazan léxicamente el valor de `this`.

### Ejemplo

```javascript
const add = (a, b) => a + b;
console.log(add(2, 3)); // Salida: 5
```

## Template Literals 

Los template literals permiten una interpolación de cadenas y cadenas multilínea más fácil. Utilizan comillas invertidas (`) en lugar de comillas simples o dobles.

### Ejemplo

```javascript
const name = "Jane";
console.log(`¡Hola, ${name}!`);
// Salida: ¡Hola, Jane!
```

La asignación por desestructuración te permite extraer valores de arreglos u objetos y asignarlos a variables de una manera más concisa.

### Ejemplo

```javascript
const person = { name: "Jane", age: 30 };
const { name, age } = person;
console.log(name, age); // Salida: Jane 30
```
